    • How to install Zola

    Although initially I started using Windows, I already have my repos setup in WSL in Linux, so, covering that install.

    Download

    This process is going to download the binary for Zola.

    • Go over to the Zola site, and look for the "unknown linux gnu" GZip, copy the link.
    • In WSL, CD over to /usr/local/bin
    • Switch to root
    • Run curl -L https://github.com/getzola/zola/releases/latest/download/zola-x86_64-unknown-linux-gnu.tar.gz -o zola.tar.gz replacing the URL with what you've copied.
    • Run tar -xzf zola.tar.gz
    • Run chmod +x zola
    • Run rm zola.tar.gz for proper cleanup
